## Image Specifications

All images should be generated with the following specifications:
- **Resolution:** 8K (7680x4320) or minimum 4K (3840x2160)
- **Aspect Ratio:** 16:9 for hero images, 4:3 for product images, 1:1 for icons
- **Style:** Professional industrial photography
- **Lighting:** Dramatic, high contrast, industrial lighting
- **Color Grading:** Realistic, slightly desaturated for industrial feel
- **Quality:** Ultra-realistic, high detail, professional grade

## Usage Notes

1. Replace placeholder divs in components with actual `<Image>` components once images are generated
2. Optimize all images for web (WebP format recommended)
3. Use Next.js Image component for automatic optimization
4. Maintain consistent color grading across all images
5. Ensure all images reflect the high-stakes, professional industrial aesthetic
